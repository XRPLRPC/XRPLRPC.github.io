quick start

Two minutes
to first call.

Get an API key, make your first request. No OAuth, no SDK, no ceremony. Standard rippled JSON-RPC over HTTPS.

01
Get your API key
Sign up via RapidAPI or email us. Your key is issued instantly. Free trial — no card required.
02
Make your first request
Send a POST to our endpoint with your X-API-Key header. All standard rippled methods work out of the box.
03
Switch from public nodes
Replace s1.ripple.com or xrpl.ws with our endpoint. One line change. No more rate limits.
example request 
# Check account info
curl -X POST https://xrplrpc.github.io/rpc \
  -H "X-API-Key: your-key-here" \
  -H "Content-Type: application/json" \
  -d '{
  "method": "account_info",
  "params": [{
    "account": "rYourWalletAddress",
    "ledger_index": "validated"
  }]
}'

# WebSocket — real-time subscription
const ws = new WebSocket("wss://xrplrpc.github.io/ws");
ws.send(JSON.stringify({
  "command": "subscribe",
  "accounts": ["rYourAddress"]
}));
